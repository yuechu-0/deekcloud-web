三行代码，极速切流

无需修改您的底层业务逻辑。全面兼容 OpenAI SDK 格式，您只需修改 base_url 和 api_key，即可瞬间无缝切换到 DeekCloud 的高速 API 节点。

完美兼容各大 Telegram 开源 Bot 及客户端框架 
from openai import OpenAI

# 只需替换为您在 DeekCloud 获取的专属密钥与节点地址
client = OpenAI(
    api_key="sk-your-deekcloud-key",
    base_url="https://api.deekcloud.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat", # 或 "gemini-pro" / "claude-3-opus"
    messages=[
        {"role": "user", "content": "写一段Python并发处理代码"}
    ]
)

print(response.choices[0].message.content)_
